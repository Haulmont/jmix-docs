== Creating Docker Image
You can create an OCI image.

```
project-root$ gradle bootBuildImage
```

You can find more information in link:https://docs.spring.io/spring-boot/docs/2.4.1/gradle-plugin/reference/htmlsingle/#build-image[Spring Boot Documentation^]
